# C-Testing-Tool
A Simple testing tool for C programming language

# Pre-requisite
* GNU Compiler Collection **GCC**

# Advantages
* Easy to use
* No dependency issues

# How to get started
```sh
# Configure the makefile depending on your project
# Then
> make
> ./example.exe
```
