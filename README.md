# C-Testing-Tool
A Simple testing tool for C programming language

# Pre-requisite
* GNU Compiler Collection **GCC

# Advantages
* Easy to use
* No dependency issues

# How to get started
```sh
> gcc -L ./lib -o test main.c -ltesting
> ./test.exe
```
